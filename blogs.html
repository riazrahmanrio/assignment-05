Question-1: What are the different ways to select an element in the DOM?

There are many ways to select element in the DOM like getElementById, getElementsByclassName, getElementByIdTagName, querySelectorAll, querySelector, etc.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML is the full html code inside that particular element where innerText is just the visible text inside that particular element and innerContent is all the text content with the element but without html format.

Question-3: What is event delegation in the DOM?

Event delegation is method by which you can use event listener once for every child inside a parent div. You dont need to use event listener for each child nodes.

Question-4: What is event bubbling in the DOM?

Event bubbling is a method that propagates the even listener until the propagation reaches the document through the parent divs one by one.

Question-5: How do you create, add, and remove elements using JavaScript?

To create an element we need to use document.createElement and to add the element we need to use document.parent.appendChild and to remove an element we need to use document.parent.removeChild
